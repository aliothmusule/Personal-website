Objetivos del Camino Full Stack

  • Aprender HTML y CSS básicos.
  • Desarrollar un sitio web estático.
  • Conocer los fundamentos de JavaScript.
  • Crear aplicaciones web dinámicas con JavaScript.
  • Aprender Git y control de versiones.
  • Manejar preprocesadores como Sass o LESS.
  • Dominar frameworks front-end como React o Angular.
  • Conocer bases de datos relacionales (MySQL, PostgreSQL).
  • Aprender Node.js y Express para backend.
  • Desplegar aplicaciones en la nube (AWS, Heroku).
  • Implementar autenticación y seguridad en las aplicaciones.
  • Participar en proyectos colaborativos y open-source.
